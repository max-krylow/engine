# WaSaby template engine

[![Build Status](https://travis-ci.org/max-krylow/engine.svg?branch=diploma)](https://travis-ci.org/max-krylow/engine)

## Install

```bash
npm install
```

## Documentation
```bash
npm run doc
```

## Build

```bash
npm run build
```

## Test

Run all unit tests
```bash
npm run test
```

Take coverage
```bash
npm run coverage
```
